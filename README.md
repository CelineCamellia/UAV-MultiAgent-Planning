# UAV-MultiAgent-Planning

UAV-MultiAgent-Planning 是一个面向无人机任务规划与仿真展示场景的 Python 智能系统项目。

项目主要围绕无人机巡检、路径规划、禁飞区避让和动态重规划展开，目标是将路径规划算法、规则化任务流程、执行日志和可视化展示整合成一个可运行、可演示、可解释的任务规划系统。

## 功能概览

* 支持结构化任务输入
* 支持二维栅格地图建模
* 支持起点、终点、障碍物和禁飞区配置
* 基于 A* 算法进行路径规划
* 支持路径轨迹可视化展示
* 支持动态禁飞区场景下的路径重规划
* 支持执行日志展示规划状态、触发原因和安全校验结果
* 支持三维多无人机轨迹仿真展示

## 快速启动

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动页面

```bash
streamlit run app.py
```

启动后，可以在页面中设置无人机起点、目标点、障碍物和禁飞区，并查看路径规划、动态重规划和轨迹可视化结果。

## 设计思路

本项目不是实际无人机飞控系统，也不是复杂强化学习训练平台，而是面向任务规划和仿真展示的轻量智能系统。

系统整体流程如下：

```text
任务输入
→ 地图建模
→ 障碍物 / 禁飞区配置
→ A* 路径规划
→ 路径安全检查
→ 动态禁飞区触发
→ 路径重规划
→ 轨迹可视化
→ 日志输出
```

## 核心模块

### A* 路径规划

系统在二维栅格地图中搜索从起点到目标点的可行路径，并绕开障碍物和禁飞区。

### 动态重规划

当动态禁飞区出现或原路径受阻时，系统重新生成可行路径，并通过日志展示触发原因和规划状态。

### 多无人机轨迹展示

系统支持多无人机轨迹仿真展示，用于直观呈现多架无人机在任务场景下的飞行轨迹、避障过程和任务执行结果。

## 演示场景

* 单无人机避障路径规划
* 动态禁飞区触发后的路径重规划
* 多无人机轨迹仿真展示
* 路径变化与任务执行日志展示
