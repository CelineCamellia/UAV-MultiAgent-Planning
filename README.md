# UAV Agent

无人机多智能体任务规划与三维协同仿真系统。

## 运行方式

只需要一个入口：

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

系统功能：

- 运行任务规划
- 生成 / 刷新三维仿真 GIF
- 一键运行完整流程
- 清空页面状态 / 重新运行

三维仿真模块已集成到主系统入口。
二维路径规划结果会同步传入三维仿真模块，三维动图使用当前任务的起点、终点、路径和障碍物场景。
目标点对应地面目标区域，飞行高度由统一巡航高度控制；障碍物和动态禁飞区以三维空域体形式显示。
三维轨迹包含起飞、巡航、下降和到达地面目标点后的短暂停留，确保三架无人机均到达终点后动画结束。
侧边栏的场景编号用于切换不同障碍物布局；三维动画帧数控制 GIF 的帧数，数值越大动画越平滑，但生成时间越长。

## 项目概述

本系统以 Streamlit 作为统一控制台，上层完成自然语言任务输入、任务解析、路径规划和安全校验，下层集成三维多无人机协同仿真模块，支持在同一页面内生成编队飞行与避障 GIF。
